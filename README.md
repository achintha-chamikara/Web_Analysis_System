# Website Analysis System 🌐

A Python-based website analysis tool that crawls websites,
stores data, and generates insights with visualizations and AI.

## Features
- Web crawler & scraper
- SQLite data storage
- Summary statistics
- Dashboard & visualizations (coming soon)
- Forecasting module (coming soon)
- AI advisor & report generator (coming soon)

## Tech Stack
- Python, BeautifulSoup, Scrapy
- SQLite, Pandas
- Plotly Dash
- AI integration

## Setup
```bash
pip install -r requirements.txt
python main.py
```
